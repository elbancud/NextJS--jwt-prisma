## Prisma config

```
to iniitalize primsa client
> npx prisma init

to migrate db & schema
> npx prisma db push
```

## Bells & Whistles

```
- mysql | Linux
    - use mysql_upgrade
    > sudo /opt/lampp/bin/mysql_upgrade
    - .env
        - make sure to start DATABASE_URL with "mysql://"
        - make sure same port
```
